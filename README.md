# Моё выздоровление

Telegram-бот для ежедневной работы по программе 12 шагов.

## Что есть в этой версии

- регистрация пользователя: псевдоним, дата трезвости, часовой пояс
- утренний настрой
- вечерняя инвентаризация
- инвентаризация страхов
- инвентаризация обид
- справочник духовных принципов
- статистика «Моё выздоровление»
- JSON-хранилища в папке data
